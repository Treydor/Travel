## Travel Project
A simulated travel planning website by [Noah Rogers](http://treydor.github.io/).

This project has been created using [Jekyll](http://jekyllrb.com).

## Jekyll Setup


#### Installing Jekyll
```
gem install jekyll
```

#### Building a new site
```
jekyll new yourname
```

#### Changing directories
```
cd yourname
```

#### Starting the server
```
jekyll serve
```

#### Visiting the server
```
localhost:4000
```
